# 机动车保险数据变量说明

本说明基于原始数据文档翻译整理，包含两个主要数据表：**保单信息表** 和 **理赔类型示例表**。

## 表1：保单信息变量

| 变量名 | 中文描述 |
|--------|----------|
| ID | 为每个年度保单分配的内部识别号。每个投保人在数据集中可能有多条记录，代表产品的不同年度。 |
| Date_start_contract | 投保人合同的开始日期（日/月/年）。 |
| Date_last_renewal | 上次合同续保的日期（日/月/年）。 |
| Date_next_renewal | 下一次合同续保的日期（日/月/年）。 |
| Distribution_channel | 合同签订渠道分类：0 代表代理人，1 代表保险经纪人。 |
| Date_birth | 保单中被保险人的出生日期（日/月/年）。 |
| Date_driving_licence | 被保险人驾驶执照的签发日期（日/月/年）。 |
| Seniority | 被保险人与保险机构关联的总年数，表示其资历水平。 |
| Policies_in_force | 在参考期间内，被保险人持有的有效保单总数。 |
| Max_policies | 被保险人曾与保险机构同时持有的最大保单数量。 |
| Max_products | 被保险人曾同时持有的最多产品数量。 |
| Lapse | 在当前年度内，客户取消或因未付款而失效的保单数量（不包括被其他保单替代的情况）。 |
| Date_lapse | 失效日期，即合同终止日期（日/月/年）。 |
| Payment | 参考保单的最近支付方式：1 代表半年期行政流程，0 代表年度支付。 |
| Premium | 当前年度保单的净保费金额。 |
| Cost_claims_year | 当前年度该保险保单产生的总理赔成本。 |
| N_claims_year | 当前年度该保险保单发生的理赔总次数。 |
| N_claims_history | 该保险保单整个存续期间发生的理赔总次数。 |
| R_Claims_history | 特定保单的理赔次数与该保单有效总年数（整年）的比率，表示保单的历史理赔频率。 |
| Type_risk | 保单关联的风险类型：1=摩托车，2=面包车，3=乘用车，4=农用车。 |
| Area | 二分类变量表示区域：0=农村，1=城市（超过3万居民）的交通状况。 |
| Second_driver | 1=有多名常规驾驶员，0=仅一名驾驶员申报。 |
| Year_matriculation | 车辆注册年份（年）。 |
| Power | 车辆功率，以马力为单位。 |
| Cylinder_capacity | 发动机排量（汽缸容量）。 |
| Value_vehicle | 截至2019年12月31日的车辆市场价值。 |
| N_doors | 车辆车门数量。 |
| Type_fuel | 车辆使用的能源类型：汽油（P）或柴油（D）。 |
| Length | 车辆长度（米）。 |
| Weight | 车辆重量（千克）。 |

## 表2：按类型划分的理赔示例

| 变量名 | 中文描述 |
|--------|----------|
| ID | 为每个年度保单分配的内部识别号。每个投保人在数据集中可能有多条记录，代表产品的不同年度。 |
| Cost_claims_year | 当前年度该保险保单产生的总理赔成本。 |
| Cost_claims_by_type | 按类型划分的理赔总成本。 |
| Claims_type | 理赔类型，共九种：道路救援、玻璃破损、投诉、过失、盗窃、火灾、全险、人身伤害、其他。 |

**说明：** 本文件根据原始英文变量说明翻译整理，所有日期格式均为日/月/年。若在数据分析或建模过程中需要进一步解释，请参考原始数据字典。
